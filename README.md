# 🛡️ Phishing Detector GUI  

🚀 A simple Python-based phishing URL detector with a GUI using Tkinter.

## 🌟 Features  
✅ Detects phishing words in URLs  
✅ Checks domain WHOIS information  
✅ Identifies shortened URLs (bit.ly, tinyurl)  
✅ Easy-to-use GUI
